# Оптимизация маршрутов доставки товаров на основе моделей глубокого обучения

Выпускная квалификационная работа

## Цель работы

**Сравнительный анализ эффективности классических, нейросетевых методов и методов обучения с подкреплением для оптимизации логистических маршрутов** на реальных данных дорожной сети Московского региона и складов Ozon. Исследование проводится в статической и динамической постановках (с учётом пробок, блокировок, временного профиля загрузки дорог).

### Основные задачи

- Обзор эволюции задач маршрутизации.
- Сбор и предобработка реальных данных: граф дорог Москвы и МО (OSMnx), геокодирование 29 складов Ozon, построение асимметричных матриц расстояний.
- Моделирование дорожных сценариев: «Без шума», «Пробки», «Блокировки+Пробки», «Умеренные условия».
- Реализация трёх классов алгоритмов:
  - **Классические** — жадный, 2‑OPT, Tabu Search, генетический алгоритм, точное ДП (Беллман–Хелд–Карп).
  - **Нейросетевые** — Pointer Network, Attention Model, графовые сети (GNN), MatNet, MatPOENet.
  - **Обучение с подкреплением** — Policy Gradient, A2C, PPO, DQN, Double DQN, SAC (базовые и улучшенные версии).
- Эксперименты в статике (12 и 29 складов, 4 сценария) и динамике (смена матриц по часам).
- Оценка по длине/времени маршрута, устойчивости к шуму, масштабируемости, стабильности обучения.
- Формулировка практических рекомендаций для логистических компаний.
